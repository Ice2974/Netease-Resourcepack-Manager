# Netease Resourcepack Manager

`Netease Resourcepack Manager` 是面向 Windows 的网易 Minecraft 基岩版资源包管理工具（桌面版）。它仅处理本地文件，把资源包替换流程做成可视化、可回滚、可追踪日志的简单操作。

## 适用范围

- **系统**：Windows 10 / Windows 11
- **目标游戏**：网易 MCBE
- **启动器限制**：仅限 “Java经典版” 启动器
- **使用前提**：目标资源包已下载到本地 `packcache` 目录

## 功能一览

- 自动扫描：`%AppData%\MinecraftPE_Netease\packcache`
- 展示资源包名称与图标
- 拖拽或选择 `.zip` / `.mcpack` 导入
- Bedrock 资源包基础合法性校验（含外层目录结构）
- 三种替换模式：全量替换、覆盖合并、仅新增
- 替换时始终保留目标资源包原 `manifest.json`
- 替换前自动备份
- 替换失败自动回滚
- 支持一键回滚到最近一次替换前状态
- 删除资源包：单个删除、全部删除、批量选择后按 `Delete` 删除
- 删除操作将资源包移入 Windows 回收站，便于误删恢复
- 日志记录与日志目录查看
- 跟随 Windows「默认应用模式」自动切换浅色 / 深色主题

## 使用方法

如果你下载的是发布版压缩包：

1. 解压整个发布包到任意目录。
2. 双击 `NeteaseResourcepackManager.exe` 启动。
3. 在主界面确认能看到目标资源包（若游戏未下载资源包，请先进入对应服务器完成下载）。
4. 双击目标资源包进入导入界面。
5. 拖入或选择你的 `.zip` / `.mcpack` 文件。
6. 选择需要的替换模式（详见下节）。
7. 校验通过后点击「执行替换」。
8. 成功后按提示重新进入服务器生效。

若替换失败，工具会自动回滚到替换前状态，并记录日志方便排查。

## 替换模式说明

| 模式 | 行为 | 适用场景 |
|---|---|---|
| **全量替换**（默认） | 清空目标资源包目录中除原 `manifest.json` 外的所有内容，再复制导入包内容。 | 想完全用导入包内容替换原资源包。 |
| **覆盖合并** | 不清空目标目录，导入包文件按路径覆盖同名文件，目标中独有的旧文件保留。 | 只想覆盖部分内容，同时保留原资源包的其他文件。 |
| **仅新增** | 不清空目标目录，仅写入目标中不存在的文件，已存在的同名文件/目录跳过。 | 只想补充目标资源包缺少的文件，不覆盖任何现有内容。 |

三种模式都会**保留目标资源包原 `manifest.json`**，不会用导入包的 `manifest.json` 覆盖。

## 删除与回收站说明

本工具提供三种删除入口：

- **单个删除**：选中资源包后，点击卡片右侧的垃圾桶图标，确认后删除。
- **全部删除**：点击主界面右上角「全部删除」按钮，确认后删除列表中所有资源包。
- **批量删除**：在列表中用鼠标多选资源包，按键盘 `Delete` 键，确认后批量删除。

删除操作会调用 Windows 系统 API 将资源包目录**移入回收站**，不会立即永久删除。若误删，可打开回收站还原。

注意：

- 按 `Delete` 键删除仅在主界面生效；当焦点在输入框、文本框等可编辑控件中时，不会触发资源包删除。
- 未选中任何资源包时按 `Delete` 键不执行操作。
- 删除前会弹出确认对话框，避免误操作。

## 数据与文件位置

程序运行数据会写到：

- `%AppData%\NeteaseResourcepackManager\backups`
- `%AppData%\NeteaseResourcepackManager\logs`
- `%AppData%\NeteaseResourcepackManager\temp`

资源包扫描目录：

- `%AppData%\MinecraftPE_Netease\packcache`

## 常见问题

### 1）提示文件无法删除/覆盖
通常是游戏或启动器仍在占用文件。请先关闭 Minecraft / 启动器后重试。

### 2）导入文件被判定不合法
仅支持 `.zip` / `.mcpack`，并且压缩包内应包含可解析的 `manifest.json`。外层多一层文件夹是支持的。

### 3）替换后没有生效
请按提示重新进入服务器；必要时确认目标是否是你当前使用的云材质位。

### 4）如何恢复到替换前
替换成功后，在结果区域点击「回滚到替换前」。

### 5）删除的资源包还能找回吗
删除操作会移入 Windows 回收站，可在回收站中还原。如果已经清空回收站，则无法通过本工具恢复。

### 6）界面没有跟随系统变深/变浅
本工具读取 Windows「设置 > 个性化 > 颜色 > 选择默认应用模式」。修改系统设置后重新启动工具即可生效。部分旧版本 Windows 可能不支持深色标题栏，工具会降级为系统默认样式。

## 安全说明

- 工具不会联网上传你的文件。
- 工具不会修改目标资源包原 `manifest.json`。
- 替换前会自动备份，失败时自动回滚。
- 删除操作会移入回收站，保留一次撤销机会。
- 所有关键步骤都有日志，便于排查。
